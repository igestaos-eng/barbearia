# Barbershop System

## Project Documentation

### Introduction
The Barbershop System is a comprehensive solution to manage appointments, customer data, and barbershop operations efficiently.

### Features
- **Appointment Management**: Schedule and manage appointments with ease.
- **Customer Database**: Maintain detailed records of customers.
- **Reporting**: Generate reports on appointments and customer visit history.

### Technologies Used
- **Frontend**: React.js
- **Backend**: Node.js, Express
- **Database**: MongoDB

## Setup Instructions

### Prerequisites
Make sure you have the following installed:
- Node.js (v14 or later)
- MongoDB

### Installation Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/igestaos-eng/barbearia.git
   cd barbearia
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start MongoDB server:
   ```bash
   mongod
   ```
4. Run the application:
   ```bash
   npm start
   ```

### Usage
- Access the application at `http://localhost:3000`.
- Create your first appointment through the dashboard.
